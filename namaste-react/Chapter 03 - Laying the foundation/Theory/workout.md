### Role of type attribute in script tag ? What options can I use there ?
##### It indicates the type of script between script tags by identifying the content
value can be one among the following
* not set(default), an empty string or JavaScript MIME type ==> contains JavaScript
* importmap ==> a JSON object used to control how the browser resolves module specifiers
* module ==> code will be treated as JavaScript module
    * require the use of CORS protocol for cross-origin fetching
* speculationrules ==> a JSON object determining which resources should be prefetched or prerendered by the browser
